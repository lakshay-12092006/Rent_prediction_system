# 🏠 Rent Prediction System

A Machine Learning-based **Rent Prediction System** that predicts the estimated rent of a property based on various features such as location, area, number of bedrooms, bathrooms, furnishing status, and other property details. The project provides quick and accurate rent predictions through an easy-to-use web interface.

## 📌 Features

- Predicts estimated house rent instantly

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Rent-Prediction-System.git
```

### Navigate to the Project Folder

```bash
cd Rent-Prediction-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
